# SmartDoc-AI: 智能文档解析与多模态比对系统

SmartDoc-AI 是一款集成了传统文档解析技术与本地大语言模型（LLM）的自动化审计工具。它能够高效处理 PDF 和 Word 文档，实现文档的结构化转换，并基于语义理解生成精准的差异分析报告。

## 🌟 核心功能

- **多模态结构化解析**：支持 PDF 和 Word 文档深度扫描，精准提取多级标题及复杂表格，并统一转化为标准 Markdown 格式。
- **语义级差异算法**：利用高性能文本比对算法，通过语义清理技术消除格式干扰，直击文档核心变动。
- **多维可视化界面**：自动生成侧边栏对比 HTML 报告。用户可同步查阅原始文档图像与高亮文本差异，实现视觉与逻辑的双重校验。
- **离线 AI 审计报告**：基于本地部署的推理引擎，针对关键技术参数、合同条款的变动生成详尽的中文审计摘要，确保数据 100% 离线隐私安全。
- **高性能资源调度**：采用核心组件懒加载技术，在保证高精度分析的同时，实现本地硬件资源的最优利用。

## 🛠️ 技术栈

- **文档解析引擎**：PyMuPDF (图像渲染)、pdfplumber (高精度表格提取)、python-docx (Word 处理)
- **差异比对算法**：diff-match-patch (语义级文本对比)
- **本地 AI 引擎**：llama-cpp-python (支持本地硬件加速的推理后端)
- **终端交互界面**：Rich (工业级控制台美化框架)

## 🚀 快速上手

### 1. 环境准备
请确保系统已安装 Python 3.10 或更高版本，并配置好 C++ 编译环境（以支持底层大模型加速库的编译安装）。

### 2. 安装依赖
在项目根目录下，使用标准包管理工具安装所需依赖：
```bash
pip install -r requirements.txt
3. 模型部署
请将兼容的 GGUF 格式的大语言模型文件（推荐 Qwen2.5-7B 系列）放入项目根目录下的 models/ 文件夹中。

4. 启动程序
在终端中运行以下命令启动系统主菜单：

Bash

python main.py
📂 目录结构说明
main.py: 系统主入口，负责交互与全流程调度。

src/: 核心业务逻辑文件夹，包含解析、比对及 AI 推理模块。

input/: 待分析原始文档存放路径。

output/: 转换结果、可视化 HTML 及 AI 审计报告的导出路径。

models/: 本地大模型权重文件的存放路径。
